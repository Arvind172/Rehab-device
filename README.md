# Real-Time Motion Tracking Dashboard with 3D Visualization

A real-time rehabilitation motion tracking system built using React, WebSockets, and React Three Fiber. The project streams live motion data from external hardware devices and visualizes skeletal movement through an interactive 3D model for rehabilitation and motion analysis applications.

## Features

- Real-time motion data streaming using WebSockets
- Interactive 3D skeletal visualization
- Live joint movement rendering
- Hierarchical bone transformation system
- High-frequency real-time updates
- Modular and scalable React architecture

## Tech Stack

### Frontend
- React.js
- JavaScript
- React Three Fiber
- Three.js

### Real-Time Communication
- WebSockets

## System Functionality

- Streams continuous motion data from external devices
- Maps joint-angle inputs to skeletal movements
- Updates 3D animations in real time
- Handles high-frequency streaming efficiently
